This portfolio is a personal website to present my professional profile, skills, experiences, and projects. It includes sections such as About Me, Experience, Projects, and Contact Information. The portfolio is designed to be responsive and user-friendly.

## Installation

To set up the project locally, follow these steps:

1. **Clone the repository**
   ```bash
   git clone https://github.com/fosterb1/portfolio.git
   ```

2. **Navigate to the project directory**
   ```bash
   cd portfolio
   ```

3. **Install dependencies**
   Ensure you have Node.js and npm installed. Then run:
   ```bash
   npm install
   ```

## Usage

To run the project locally, use the following command:

```bash
npm start
```

Open your browser and navigate to `http://localhost:3000` to view the portfolio.

## Features

- **Responsive Design:** The portfolio is fully responsive and adapts to different screen sizes.
- **Interactive Navigation:** Includes both desktop and mobile-friendly navigation menus.
- **Dynamic Content:** Sections dynamically display content about my experience, projects, and contact information.
- **Social Media Integration:** Links to my social media profiles for easy connection.
